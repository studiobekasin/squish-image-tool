# Squish — Image Compressor & Resizer

A simple, fast, client-side image compressor and resizer. Everything runs locally in your browser — no uploads, no servers.

## Features

- Drag & drop or click to add images (batch supported)
- Adjustable quality (1-100%)
- Resize with aspect ratio lock
- Output format: Original, JPG, PNG, WebP
- Individual or bulk download (ZIP)
- File size savings displayed per image

## Usage

Just open `index.html` in your browser. No build step needed.
